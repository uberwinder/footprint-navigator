DATA TO CAPTURE:
- question: the user message that prompted 
  this response
- response: full response text
- rating: "up" or "down"
- comment: optional text from thumbs down
- document_name: active document filename
- page_number: current page number
- confidence: confidence value from response
- model: model name from response metadata
- timestamp: ISO string

POST to /pdf-api/chat-rating
